## Overview

GenUI is a sample app that demonstrates the full A2UI pipeline — from user prompt to native SwiftUI interface. The user enters a natural language query, the LLM generates an interactive UI by composing widgets from the app's component catalog, and the user can tap on the rendered cards to continue the conversation.

@Row {
    @Column(size: 2) {
        ### Conversational UI Generation

        The user sends a prompt like "Plan a trip to Japan!" and the
        agent responds with a rich, interactive surface — image cards,
        destination pickers, and detail views — all rendered natively
        via ``A2UIRendererView``.

        When the user taps a card, the ``ResolvedAction`` is sent back
        to the agent, and the conversation continues with new UI.
    }

    @Column {
        ```swift
        let client = try await A2AClient.fromAgentCardURL(cardURL)

        for try await event in client.sendTextStream(prompt) {
            if case .result(let result) = event {
                try manager.processMessages(result.messages)
            }
        }
        ```
    }
}

@Row {
    @Column {
        ```swift
        A2UIRendererView(manager: manager) { action in
            Task {
                try await client.sendAction(
                    action, surfaceId: "main"
                )
            }
        }
        ```
    }

    @Column(size: 2) {
        ### Two-Way Interaction

        A2UI's action system enables a closed-loop conversation.
        The agent generates UI with embedded actions; when the user
        interacts, the renderer resolves all data-bound values via
        ``ResolvedAction`` and sends the full context back to the agent
        for the next turn.
    }
}

### Live Streaming

GenUI uses ``A2AClient`` with SSE streaming (`message/stream`) for
real-time UI updates. As the agent processes the request, status
updates flow through ``StreamEvent`` and the surface builds
incrementally via ``SurfaceManager``.

```swift
for try await event in client.sendTextStream("Plan a trip to Japan!") {
    switch event {
    case .status(let state, let text, _, _, _):
        statusMessage = text ?? state.rawValue
    case .result(let result):
        try manager.processMessages(result.messages)
    }
}
```
